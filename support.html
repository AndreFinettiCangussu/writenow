What is Write Now

Write Now is a lightweight macOS tool that turns your voice into text in real time and automatically pastes it into the active app. Perfect for email, chats, notes, and any text field.

Key features
  • Global shortcut: hold a modifier (default ⌥ Option) to speak.
  • Accurate transcription with punctuation (when available).
  • Automatically paste into the current app, or copy to clipboard.
  • Voice “shortcuts” (e.g., “inserisci shortcut firma”); the trigger phrase is customizable.
  • Personal dictionary with automatic learning of recurring terms.
  • File transcription: MP3, M4A, AAC, WAV, FLAC, OGG, OPUS, AIFF, CAF.
  • On‑device engines (Whisper, Sherpa) in Premium; Apple Speech included in Free.
System requirements
  • macOS 13 Ventura or later (macOS 14+ recommended).
  • Microphone and Accessibility permissions enabled.
  • Internet connection only to download models or system assets, when needed.
Privacy: Whisper and Sherpa run entirely on device (Premium). Apple Speech may use system services depending on language and availability. Audio is never sent to third parties without your consent.

Quick Start

  1. Install and launch Write Now.
  2. Grant access to Microphone and Accessibility when prompted.
  3. Open Settings to choose language, shortcut, and paste behavior.
  4. Hold ⌥ (default), speak, then release — your text is pasted automatically.
Status indicators
  • Listening: a dark pill with the wave; the mic is active.
  • Transcribing…: converting audio to text.
  • Pasting: inserting text into the current app.
  • Error: click to read details.
Tips
  • Select the correct language in Settings.
  • Speak clearly in a quiet environment.
  • Dictate punctuation when needed (e.g., “comma”, “period”).

Transcription engines

Apple Speech (Free)

Built into macOS. Fast setup, no model downloads. Processing may be on device or use Apple services depending on language and availability.

Whisper (Premium)
  • Fully on device, no data leaves your Mac.
  • Requires downloading a model (e.g., small/medium/large‑v3‑turbo/large‑v3).
  • Models are stored in ~/Library/Application Support/Write_Now/Models/.
Sherpa (Premium)
  • On device; streaming variants (Zipformer/Paraformer) and offline (NeMo).
  • Models are stored in ~/Library/Application Support/Write_Now/SherpaModels/.
  • Some models may output all‑caps text (normalized by the app).

Permissions

Microphone

Required to record and transcribe your voice.

  1. System Settings → Privacy & Security → Microphone.
  2. Enable Write Now and restart the app if needed.
Accessibility

Required to paste or type text automatically into the active app.

  1. System Settings → Privacy & Security → Accessibility.
  2. Enable Write Now (unlock the padlock if requested).

Data & privacy

  • No account and no tracking.
  • On‑device audio with Whisper/Sherpa (Premium). Apple Speech follows system policies.
  • Local storage under ~/Library/Application Support/Write_Now/:
    • Whisper models: .../Models/
    • Sherpa models: .../SherpaModels/
    • Personal dictionary and shortcuts: JSON files in the same folder.
  • Local statistics for word/session counts, used to enforce the Free plan’s weekly limit.

Troubleshooting

It doesn’t hear me / Empty transcription
  • Check the Microphone permission.
  • Verify your input device in System Settings → Sound → Input.
  • Reduce background noise or adjust mic distance.
Text doesn’t paste automatically
  • Enable the Accessibility permission.
  • Make sure the cursor is in an editable text field.
  • Check “Automatic paste” options in Settings.
Model download seems stuck
  • Keep the app in the foreground until the download finishes.
  • Check your connection and try again.
  • Restart the app to resume the download.
Shortcut conflicts
  • Change the global shortcut in Write Now’s Settings.
  • Avoid combinations used by macOS (e.g., Spotlight) or other apps.
Low accuracy / punctuation issues
  • Select the correct language in Settings.
  • Speak at a natural pace and dictate punctuation when needed.

FAQ

Does Write Now work offline?

Yes — with Whisper and Sherpa (Premium), everything runs on device. Apple Speech may use Apple services depending on language and system availability.

Which audio formats are supported for file transcription?

MP3, M4A, AAC, WAV, FLAC, OGG, OPUS, AIFF, and CAF.

Can I disable automatic paste?

Yes. In Settings you can choose automatic paste, copy to clipboard, or preview before pasting.

How do voice “shortcuts” work?

Set a trigger phrase (default in Italian: “inserisci shortcut”), then assign shortcut names to expand into text. Example: “inserisci shortcut firma”.

How do I restore purchases?

From Settings → Premium → “Restore purchases”.
